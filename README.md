# ЦПМС КР — Центр подготовки молодёжных сборных

## Запуск проекта

```bash
npm install
npm run dev
```

Откройте http://localhost:5173

## Структура

```
src/
├── components/
│   ├── layout/     # Header, Footer
│   ├── sections/   # HeroSection, CarouselSection, MapSection
│   └── ui/         # Shared UI components
├── context/        # AppContext (global state)
├── data/           # appData.js, translations.js
└── pages/          # All pages
```

## Функционал

### Публичный сайт
- 3D глобус Кыргызстана (Three.js) — кликабельные регионы
- Карусель с реальными фото
- Анимации (framer-motion)
- Мультиязычность: РУ / КЫ / EN
- Форма заявки с загрузкой фото

### Административная панель (/admin)
- Управление спортсменами (CRUD + фото)
- Управление тренерами, соревнованиями
- Одобрение/отклонение заявок
- Финансовые графики
- Экспорт CSV отчётов

### Портал сотрудников (/staff)
- 10 отделов с отдельными аккаунтами
- Отметка посещаемости
- Система задач с приоритетами
- Общий файловый менеджер
- Чат между сотрудниками
- Встроенный калькулятор (бухгалтерия)

## Технологии
- React 18 + Vite
- React Router v6
- Framer Motion (анимации)
- Three.js (3D карта)
- Recharts (графики)
- Tailwind CSS
- Lucide React (иконки)
